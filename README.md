## 함께 번역하기 — DLG 에디터

패치에 포함된 **아카넘 DLG 에디터** (`tools/arcanum_dlg_editor.html`)를 사용하면 누구나 대화 파일을 직접 편집할 수 있습니다. 프로그래밍 지식 없이도 아카넘의 NPC 대사를 수정하고 번역에 기여할 수 있습니다.

### 이게 뭔가요?

아카넘의 모든 NPC 대화는 `.dlg` 파일에 바이너리 형태로 저장되어 있습니다. 텍스트 에디터로는 열 수 없는 파일이라 지금까지는 전문 도구나 직접 스크립트를 작성해야만 편집이 가능했습니다.

**DLG 에디터**는 이 DLG 파일을 브라우저에서 바로 열어 볼 수 있게 해줍니다:
- NPC가 무슨 말을 하는지
- 플레이어가 어떤 선택지를 고를 수 있는지
- 각 선택지가 어떤 조건에서 나타나는지
- 선택 후 어디로 분기하는지

이 모든 것을 시각적으로 확인하면서 한글 텍스트를 수정하고, 수정된 DLG 파일을 저장하여 게임에 바로 적용할 수 있습니다.

### 시작하기

1. **에디터 열기** — `tools/arcanum_dlg_editor.html` 파일을 브라우저로 열기 (더블클릭)
2. **DLG 파일 열기** — 게임 폴더의 `modules/Arcanum/dlg/`에서 `.dlg` 파일을 드래그하거나 선택 (여러 파일 동시 가능)
3. **대사 탐색** — 왼쪽 목록에서 노드 선택. 빨간색 = NPC 대사, 초록색 = PC 선택지
4. **텍스트 편집** — 오른쪽 패널에서 한글 텍스트 수정
5. **저장** — `Ctrl+S` 또는 💾 버튼 → 다운로드된 파일을 게임 폴더에 덮어쓰기

### DLG 파일 구조 이해하기

아카넘의 대화 파일은 **노드(node)** 단위로 구성됩니다. 하나의 대화가 여러 노드의 연결로 이루어져 있습니다.

#### 노드 종류
| 종류 | 설명 | 표시 |
|------|------|------|
| **NPC 대사** | NPC가 말하는 텍스트 | 🔴 빨간색 |
| **PC 선택지** | 플레이어가 고를 수 있는 응답 | 🟢 초록색 |
| **특수 노드** | `G:`(인사), `E:`(끝), `T:`(감사), `K:`(전투) 등 구분자 | 회색 |

#### 노드 필드
| 필드 | 설명 | 예시 |
|------|------|------|
| **ID** | 노드 번호 | `42` |
| **텍스트** | 번역 대상 텍스트 (✏️ 편집 가능) | `안녕하세요, 여행자여.` |
| **align** | PC 지능 필터. 빈칸=NPC 대사, `5`=고지능 PC 전용, `-4`=저지능 PC 전용 | `5` |
| **cond** | 이 노드가 표시되는 조건 (아래 참조) | `me1, re41` |
| **next** | 선택 시 다음 노드 ID. `-2`=대화 종료, 빈칸=아래 PC 선택지 대기 | `101` |
| **script** | 선택 시 실행되는 스크립트 (아래 참조) | `$$200, QU1126 3` |

#### 조건(cond) 레퍼런스
| 코드 | 의미 |
|------|------|
| `me0` / `me1` | 처음 만남(0) / 이미 만남(1) |
| `re41` | 호감도 ≥ 41 (친밀) |
| `re-40` | 호감도 ≤ -40 (적대) |
| `lf5 1` | NPC 로컬 플래그 5 = 1 |
| `QA1126 1` | 퀘스트 1126 상태 ≥ 1 (활성) |
| `QB1126 3` | 퀘스트 1126 상태 ≤ 3 |
| `$$400` | 소지금 ≥ 400 |
| `in2827` | 인벤토리에 아이템 2827 보유 |
| `ni2827` | 인벤토리에 아이템 2827 없음 |
| `sk11 9` | 스킬 11 ≥ 레벨 9 |
| `ma-40` | 마법 적성 ≤ -40 (기술 성향) |

#### 스크립트(script) 레퍼런스
| 코드 | 의미 |
|------|------|
| `$$200` | 200코인 지급 |
| `$$-100` | 100코인 차감 |
| `QU1126 3` | 퀘스트 1126 상태를 3으로 설정 |
| `lf5 1` | 로컬 플래그 5를 1로 설정 |
| `rf100` | 호감도 100 추가 |

### 편집 시 주의사항

- ✏️ **텍스트만 편집하세요.** cond/script/next는 게임 로직이라 잘못 수정하면 게임이 깨질 수 있습니다.
- 🚫 **이모지, 특수문자 사용 금지.** 한글은 CP949 인코딩으로 저장됩니다.
- ⚠️ **`@pcname@`을 지우지 마세요.** 게임에서 플레이어 이름으로 자동 치환됩니다.
- 💾 **원본 백업 후 작업하세요!** 수정 전에 원본 DLG 파일을 따로 보관해두세요.
- 📂 **저장 위치**: 게임 폴더의 `modules/Arcanum/dlg/`에 덮어쓰면 즉시 적용됩니다.

### 특징
- **설치 불필요** — HTML 파일 하나, 브라우저만 있으면 됨
- **완전 오프라인** — 서버 연결 없음, 인터넷 불필요, 파일이 외부로 전송되지 않음
- **대화 트리 시각화** — NPC 대사/PC 선택지 구분, 조건/스크립트 표시
- **텍스트 검색** — 텍스트/노드 ID/조건으로 즉시 검색
- **다중 파일** — 여러 DLG 파일을 동시에 열어 작업 가능
- **수정 추적** — 변경된 노드에 🟡 표시, 수정 개수 실시간 카운트
- **대화 흐름 탐색** — PC 선택지 → NPC 응답, NPC 대사 → PC 선택지 바로 이동
- **키보드 단축키** — `Ctrl+S` 저장, `Ctrl+F` 검색

### 번역 기여 방법
1. DLG 에디터로 파일을 열고 번역을 수정합니다
2. 수정된 DLG 파일을 저장합니다
3. 이 저장소를 Fork하고, `modules/Arcanum/dlg/`에 수정된 파일을 넣습니다
4. [Pull Request](https://github.com/hyunjun6928-netizen/arcanum-korean-patch/pulls)를 보내주세요!

번역 품질 개선, 오타 수정, 어투 통일 등 어떤 기여든 환영합니다.
