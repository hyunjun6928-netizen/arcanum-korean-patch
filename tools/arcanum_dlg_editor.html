Arcanum: Of Steamworks and Magick Obscura — 대화 파일 편집기 v1.0

📂 DLG 파일을 드래그하거나 클릭하세요

아카넘 대화 파일(.dlg)을 열어 편집할 수 있습니다.
파일 위치: modules/Arcanum/dlg/

여러 파일을 한번에 선택할 수도 있습니다.

📖 사용 방법

1. DLG 파일 열기

게임 폴더의 modules/Arcanum/dlg/에서 DLG 파일을 선택합니다. 여러 파일을 한번에 열 수 있습니다.

2. 노드 구조 이해하기

  • NPC 대사 — NPC가 말하는 텍스트. next가 비어있고 아래에 PC 선택지가 따라옵니다.
  • PC 선택지 — 플레이어가 고를 수 있는 응답. next가 다음 NPC 노드를 가리킵니다.
  • 특수 노드 — G:(인사), E:(끝), T:(감사), K:(전투) 등 구분자.

3. 필드 설명

  • ID — 노드 번호
  • 텍스트 — 한글 번역 텍스트 (편집 가능)
  • align — PC 지능 필터. 5=고지능, -4=저지능, 빈칸=NPC 대사
  • cond — 이 노드가 표시되는 조건. me1=이미 만남, re41=호감도≥41, lf5 1=로컬플래그5=1, QA1126 1=퀘스트1126 활성
  • next — 이 선택지를 고르면 가는 NPC 노드 ID. -2=대화 종료, 빈칸=NPC 대사(아래에 PC 선택지)
  • script — 실행되는 스크립트. $$200=200코인 지급, QU1126 3=퀘스트 업데이트, lf5 1=플래그 설정

4. 편집 & 저장

텍스트를 수정하면 노드 목록에 ● 표시가 나타납니다. 파일 저장 버튼을 누르면 수정된 DLG 파일이 다운로드됩니다. 게임 폴더에 덮어쓰면 적용됩니다.

5. 주의사항

  • 텍스트만 편집하세요. cond/script/next는 게임 로직이라 잘못 수정하면 게임이 깨질 수 있습니다.
  • 한글은 CP949 인코딩으로 저장됩니다. 특수문자(이모지 등)는 사용하지 마세요.
  • @pcname@은 플레이어 이름으로 자동 치환되는 변수입니다. 지우지 마세요.
  • 원본 DLG를 백업해두고 작업하세요!

6. 고급: 조건(cond) 레퍼런스

  • me0 / me1 — 처음 만남(0) / 이미 만남(1)
  • re41 / re-40 — 호감도 ≥41(친함) / ≤-40(적대)
  • lf5 1 — NPC 로컬 플래그 5 = 1
  • QA1126 1 — 퀘스트 1126 상태 ≥ 1
  • QB1126 3 — 퀘스트 1126 상태 ≤ 3
  • QU1126 2 — 퀘스트 1126 상태 == 2
  • $$400 — 소지금 ≥ 400
  • in2827 — 인벤토리에 아이템 2827 있음
  • ni2827 — 인벤토리에 아이템 2827 없음
  • sk11 9 — 스킬 11 ≥ 9
  • ma -40 — 마법 적성 ≤ -40 (기술 성향)
